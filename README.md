# API Testing Project

## 🔍 Project Overview:
REST API Testing project using Postman and JSON Server.

## 🧪 Testing Type:
GET / POST / PUT / DELETE methods, Response Validation, Bug Reporting

## 🛠 Tools Used:
Postman / JSON Server / Excel

## 📝 Key Responsibilities:
- Designed and executed API test cases for different endpoints.  
- Validated response status codes, response body, and data accuracy.  
- Documented bugs and issues found during API testing.  
- Managed API test collections and results on GitHub for version control.  
- Uploaded test artifacts including response validation sheets and screenshots.
